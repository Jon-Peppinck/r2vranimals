has_url_prefix

Description

Does string have a URL prefix

Usage

has_url_prefix(urls)

Arguments

urls

a character vector of urls

Details

returns true if string matches an internal set of URL prefixes. Vectorised over URL parameter.

Value

a logical vector the same length as url.
