a_remove_entity

Description

Remove an A-Frame entity from the scene

Usage

a_remove_entity(id)

Arguments

id

id of the entity to be removed.

Details

Remove the entity identified by 'id' from the scene.

Value

An object that represents an A-Frame Event.
